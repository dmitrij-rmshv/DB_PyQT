Установка¶

После скачивания программы в некоторую директорию, правильным (но не обязательным) решением будет создание виртуального окружения для работы интерпретатора python. Для этого из рабочей папки вызовите команду:

$ python3 -m venv venv и активировуйте его командой:

$ source venv/bin/activate

Теперь потребуется удовлетворить зависимости (установить) ряд дополнительных библиотек:

$ pip install -r requirements.txt
