Требования к ПО¶

Программа написана на языке Python версии 3.8.10, соответственно на ОС пользователей должен быть установлен интерпретатор Python. Если таковой отсутствует его требуется установить, например с официального сайта:

https://www.python.org/downloads/

Помимо стандартной установки интерпретатора потребуется дополнительный (приведенний ниже) набор библиотек, способ установки которых описан ниже в разделе установка.

click==8.0.1

PyQt5==5.15.4

pyqt5-plugins==5.15.4.2.2

PyQt5-Qt5==5.15.2

PyQt5-sip==12.9.0

pyqt5-tools==5.15.4.3.2

python-dotenv==0.19.0

pyuic5-tool==0.0.1

qt5-applications==5.15.2.2.2

qt5-tools==5.15.2.1.2

SQLAlchemy==1.4.21

toml==0.10.2
