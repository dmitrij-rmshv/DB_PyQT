Запуск сервера
===============
Запуск сервера должен предшествовать запуску программ клиентов. Сервер должен работать непрерывно (или, по крайней мере, до отключения всех клиентов).
Осуществляется подачей команды:

``$ python3 chat_server.py [-p port]``

где необязательным параметром port можно указать отличный от 7777 по умолчанию порт.

Администратору следует сообщить IP адрес машины с запущенным сервером (а при необходимости и порт) всем потенциальным клиентам;
без этих параметров они не подключаться к серверу и не смогут начать общение
